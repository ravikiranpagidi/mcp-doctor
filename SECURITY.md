# Security Policy

MCP Doctor is an early-stage scanner and should not be treated as a complete security audit.

If you find a vulnerability in this project, please open a private security advisory if GitHub security advisories are enabled. If not, email the maintainer listed in the repository profile.

Please do not open a public issue for active exploit details.

Useful reports include:

- affected version or commit
- steps to reproduce
- expected and actual behavior
- suggested fix, if known
